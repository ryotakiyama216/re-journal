# 事業計画書: Re:Journal

## 1. エグゼクティブサマリー
Re:Journalは、AIを活用した次世代のジャーナリング（内省・日記）プラットフォームです。単なる記録ツールではなく、AIがユーザーの予定や感情に深く寄り添い、メンタルヘルスと生産性の向上を同時に実現する「デジタル・メンター」を目指します。

## 2. 解決したい課題
- **多忙による内省の欠如**: 現代人は予定に追われ、自分の感情や行動を振り返る時間が不足している。
- **日記の継続困難**: 従来の白紙の日記はハードルが高く、多くの人が三日坊主で終わってしまう。
- **孤独感とメンタルヘルス**: 誰かに話を聞いてほしいが、適切な相手がいない、あるいは気を使ってしまう。

## 3. 市場分析と可能性

### 3.1 市場規模 (Market Size)
- **メンタルヘルス・アプリ市場**: 世界のメンタルヘルス・アプリ市場は、2023年に約62億ドルと評価され、2030年までに年平均成長率(CAGR)15%以上で成長すると予測されています。
- **セルフケア・ウェルネス市場**: 「マインドフルネス」や「ウェルビーイング」への関心の高まりにより、Z世代を中心にデジタルツールへの支出が増加しています。

### 3.2 市場の可能性
- **AIによるパーソナライズ**: 従来のアプリは静的でしたが、LLM（大規模言語モデル）の登場により、個々のユーザーに最適化された対話が可能になり、市場が再定義されています。
- **B2Bへの展開**: 従業員のメンタルヘルスケアとして、法人向け福利厚生（EAP）としての導入ポテンシャルも非常に高いです。

## 4. プロダクト戦略 (v1 -> v2)
- **v1 (MVP)**: 予定の整理とAIによる行動アドバイス（実用性重視）。
- **v2 (深化)**: 感情分析、過去の対話に基づいたパーソナライズ、Stripeによるサブスクリプションモデルの導入。

## 5. 収益モデル
- **Freemiumモデル**:
  - **Free**: 基本的なジャーナリング、限定的なAI対話。
  - **Premium (月額 980円〜)**: 無制限のAI対話、高度な感情分析レポート、過去データの無制限保存、カスタムAI人格。

## 6. ロードマップ
1.  **Phase 1**: MVPリリースとユーザーフィードバックの収集。
2.  **Phase 2**: Supabase連携によるデータ永続化とパーソナライズ強化。
3.  **Phase 3**: Stripe連携によるサブスクリプション開始。
4.  **Phase 4**: モバイルアプリ化（PWA/Native）とグローバル展開。

## 7. リスクと対策
- **データプライバシー**: ユーザーの極めて個人的なデータを扱うため、SupabaseのRLSや暗号化を徹底し、プライバシーポリシーを明確にする。
- **AIの回答精度**: Difyを活用したプロンプトエンジニアリングとRAGの導入により、ハルシネーション（誤情報）を抑制し、質の高い対話を実現する。
